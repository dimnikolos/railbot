# Οδηγός Έργου (Train Game)

Αυτό το αρχείο προορίζεται για τον agent (AI) προκειμένου να κατανοήσει άμεσα τη δομή του project, τον τρόπο λειτουργίας του κώδικα και να μη χρειάζεται να ξεκινάει την ανάλυση από το μηδέν.

## Δομή του Project

Το project είναι ένα παιχνίδι όπου ο παίκτης προγραμματίζει την κίνηση ενός τρένου σε ένα πλέγμα (grid) για να μαζέψει επιβάτες.

- **`index.html`**: Περιέχει το DOM και το UI του παιχνιδιού (grid, layers για το τρένο και τους επιβάτες, καθώς και τα κουμπιά ελέγχου).
- **`activity.css`**: Περιέχει τα styles για το UI.
- **`activity.js`**: Είναι το κύριο αρχείο λογικής του παιχνιδιού (game logic). Διαχειρίζεται την κατάσταση (state), το animation της κίνησης του τρένου, το σύστημα ουράς εντολών (queue) και τη διαχείριση των επιπέδων.
- **`maze1.js`**: Περιέχει προηγούμενο ή εναλλακτικό κώδικα (κυρίως αλγόριθμο παραγωγής λαβυρίνθων). Δεν χρησιμοποιείται στην κύρια ροή του παιχνιδιού του τρένου όπως ορίζεται στο `activity.js`.
- **Εικόνες / Πόροι**: Στο root υπάρχει η εικόνα `top_down_train.png` και φάκελοι `credits` και `resource`.

## Βασική Λογική (`activity.js`)

Το παιχνίδι διαδραματίζεται σε ένα Grid (με προεπιλεγμένο μέγεθος 5x5, καθορισμένο από τη σταθερά `GRID_SIZE`).
Το τρένο ξεκινάει από μια προκαθορισμένη θέση και κατεύθυνση, π.χ. κάτω αριστερά `(x: 0, y: 4)` κοιτώντας πάνω (`dir: 0`).
- Ο παίκτης προσθέτει εντολές ('straight', 'left', 'right') σε μια ουρά.
- Όταν εκτελεστούν, το script δημιουργεί ράγες στο πλέγμα και το τρένο μετακινείται χρησιμοποιώντας CSS transforms.
- Σκοπός είναι να περάσει από όλα τα κελιά που περιέχουν επιβάτες. Όταν συλλεχθούν όλοι οι επιβάτες (`!p.collected`), το επίπεδο ολοκληρώνεται.

## Πώς δημιουργούνται τα επίπεδα

Όλα τα επίπεδα ορίζονται μέσα στη συνάρτηση **`setupLevel()`** στο αρχείο **`activity.js`**. 

Για να διαβάσετε, να επεξεργαστείτε ή να δημιουργήσετε ένα νέο επίπεδο, ακολουθήστε τα παρακάτω βήματα:

1. Βρείτε τη συνάρτηση `setupLevel()` στο `activity.js`.
2. Μέσα σε αυτή, υπάρχει μια λογική `if - else if` που ελέγχει τη μεταβλητή `level`.
3. Για να προσθέσετε ένα **νέο επίπεδο**, προσθέστε ένα νέο block κώδικα `else if (level === X) { ... }`.
4. Σε κάθε επίπεδο, η τοποθέτηση των επιβατών γίνεται καλώντας τη συνάρτηση **`addPassenger(x, y)`**.
   - Οι συντεταγμένες `x` και `y` παίρνουν τιμές από `0` έως `GRID_SIZE - 1` (δηλαδή `0` έως `4` για πλέγμα 5x5).
   - Το `(0, 0)` είναι η πάνω αριστερά γωνία, ενώ το `(0, 4)` είναι η κάτω αριστερά (όπου αρχίζει το τρένο συνήθως).

**Παράδειγμα Κώδικα για Επίπεδα:**

```javascript
    if (level === 1) {
      // Επίπεδο 1: 2 επιβάτες
      addPassenger(0, 1);
      addPassenger(3, 1);
    } else if (level === 2) {
      // Επίπεδο 2: 2 επιβάτες
      addPassenger(1, 2);
      addPassenger(4, 0);
    } else if (level === 3) {
      // Επίπεδο 3
      addPassenger(2, 2);
      addPassenger(4, 4);
    } else if (level === 4) {
      // ΠΑΡΑΔΕΙΓΜΑ ΝΕΟΥ ΕΠΙΠΕΔΟΥ 4
      addPassenger(1, 1);
      addPassenger(2, 0);
      addPassenger(3, 3);
    }
```

*Σημείωση:* Για να αλλάξετε την αρχική θέση του τρένου σε κάποιο επίπεδο (ή γενικά), μπορείτε να επέμβετε στο αντικείμενο `trainState` που αρχικοποιείται στην αρχή της `setupLevel()`, π.χ.:
`trainState = { x: 0, y: 4, dir: 0, rotation: 0 };`
(όπου `dir: 0` = Πάνω, `1` = Δεξιά, `2` = Κάτω, `3` = Αριστερά).
