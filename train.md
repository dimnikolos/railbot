# Οδηγός Έργου (Railbot)

Αυτό το αρχείο προορίζεται για τον agent (AI) προκειμένου να κατανοήσει άμεσα τη δομή του project, τον τρόπο λειτουργίας του κώδικα και να μη χρειάζεται να ξεκινάει την ανάλυση από το μηδέν.

## Δομή του Project

Το project είναι ένα παιχνίδι όπου ο παίκτης προγραμματίζει την κίνηση ενός τρένου σε ένα πλέγμα (grid) για να επισκεφτεί σταθμούς.

- **`index.html`**: Περιέχει το DOM, το UI του παιχνιδιού (grid, layers για το τρένο και τους σταθμούς, καθώς και τα κουμπιά ελέγχου) και τα audio tags για τους ήχους.
- **`activity.css`**: Περιέχει τα styles για το UI (Glassmorphism, Grid Layouts, Control Panels).
- **`activity.js`**: Είναι το κύριο αρχείο λογικής του παιχνιδιού (game logic). Διαχειρίζεται:
  - Το state (θέση, κατεύθυνση, rotation του τρένου).
  - Το animation της κίνησης του τρένου και το dynamic track generation.
  - Το σύστημα ουράς εντολών (queue) με **interactive scrubbing** (κλικ σε εντολή) και **single command delete**.
  - Την κίνηση του τρένου και τους κανόνες.
  - Αναπαραγωγή ήχων (train_move.mp3, train_whistle.mp3).
- **`data.js`**: Διαχειρίζεται την αποθήκευση και φόρτωση των δεδομένων των επιπέδων μέσω `localStorage`.
- **`builder.js`**: Υλοποιεί το In-Game Level Builder (επεξεργασία επιπέδων) που επιτρέπει τη δημιουργία σταθμών και εμποδίων απευθείας στο UI.
- **`advanced.html` & `advanced.js`**: Υλοποιούν το **Advanced Mode**, το οποίο χρησιμοποιεί ένα μεγαλύτερο πλέγμα (8x8) και ενσωματώνει τη βιβλιοθήκη **Blockly** για προγραμματισμό με τουβλάκια (συμπεριλαμβανομένων επαναλήψεων και actions όπως σφύριγμα τρένου). Η εκτέλεση του ήχου περιμένει να ολοκληρωθεί μέσω Promise (onended event).
- **`advanced_builder.js`**: Υλοποιεί τον Level Builder αποκλειστικά προσαρμοσμένο για το 8x8 πλέγμα του Advanced Mode.

## Βασική Λογική (`activity.js`)

Το παιχνίδι διαδραματίζεται σε ένα Grid (με προεπιλεγμένο μέγεθος 5x5, καθορισμένο από τη σταθερά `GRID_SIZE`).
Το τρένο ξεκινάει από μια προκαθορισμένη θέση και κατεύθυνση, π.χ. κάτω αριστερά `(x: 0, y: 4)` κοιτώντας πάνω (`dir: 0`).
- Ο παίκτης προσθέτει εντολές ('straight', 'left', 'right') σε μια ουρά. Οι εντολές εμφανίζονται γραφικά.
- Η ουρά είναι διαδραστική: κάνοντας κλικ σε μια εντολή, το πρόγραμμα "μεταπηδά" (instant simulate) σε εκείνη τη χρονική στιγμή.
- Πατώντας το **Play**, το script δημιουργεί ράγες στο πλέγμα και το τρένο μετακινείται ομαλά χρησιμοποιώντας CSS transforms.
- Το **Stop** σταματάει την κίνηση (περιμένοντας να ολοκληρωθεί το τρέχον βήμα) και επαναφέρει αμέσως το παιχνίδι στο βήμα 0.
- Σκοπός είναι να περάσει από όλα τα κελιά που περιέχουν σταθμούς (`passengers` array). Όταν το τρένο περνάει από έναν σταθμό, αυτός γίνεται γκρι (collected). Όταν επισκεφτεί όλους τους σταθμούς, το επίπεδο ολοκληρώνεται.

## Πώς δημιουργούνται τα επίπεδα

Πλέον, το παιχνίδι διαθέτει ενσωματωμένο **Level Builder**.

Για να επεξεργαστείτε ένα επίπεδο:
1. Πατήστε το εικονίδιο επεξεργασίας (μολύβι) δίπλα στο επίπεδο από το UI του παιχνιδιού.
2. Το παιχνίδι μπαίνει σε **Builder Mode**.
3. Κάνοντας κλικ στα κελιά του πλέγματος, μπορείτε να προσθέσετε Σταθμούς (Stations), Εμπόδια (Boulders) ή να τα καθαρίσετε (Blanks).
4. Πατήστε **Save Level** για να αποθηκεύσετε τις αλλαγές (αποθηκεύονται τοπικά στο browser μέσω `localStorage` και διαχειρίζονται από το `data.js`).

Πίσω από τα παρασκήνια (για developers):
- Το αντικείμενο `window.LEVELS` χωρίζεται πλέον σε `normal` (για το 5x5) και `advanced` (για το 8x8). Τα προεπιλεγμένα επίπεδα ορίζονται στο αρχείο **`data.js`** μέσα στις μεταβλητές `defaultLevels` και `advancedLevels`.
- Κάθε επίπεδο είναι ένας πίνακας με αντικείμενα που περιέχουν συντεταγμένες `x`, `y` και `type` (π.χ. `station`, `boulder`).
- Τα **`builder.js` / `advanced_builder.js`** αναλαμβάνουν το User Interface για τη δημιουργία/επεξεργασία αυτών των δεδομένων, ενώ τα **`activity.js` / `advanced.js`** αναλαμβάνουν την απεικόνιση των αντικειμένων αυτών κατά τη διάρκεια του παιχνιδιού μέσω της συνάρτησης `setupLevel()`.

*Σημείωση:* Για να αλλάξετε την αρχική θέση του τρένου, τροποποιήστε το `trainState`:
`trainState = { x: 0, y: 4, dir: 0, rotation: 0 };`
(`dir: 0` = Πάνω, `1` = Δεξιά, `2` = Κάτω, `3` = Αριστερά).
